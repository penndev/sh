# NodeJS开发手记

## nvm （nodejs 版本管理软件）

```
nvm list　　//查看目前已经安装的版本

nvm list available //显示可下载版本的部分列表

nvm install 10.15.0 //安装指定的版本的nodejs

nvm use 10.15.0 //使用指定版本的
nodejsnpm install -g cnpm --registry=https://registry.npm.taobao.org  //使用淘宝镜像
```

## npm

##### NRM （npm的源管理包）

npm install -g nrm
